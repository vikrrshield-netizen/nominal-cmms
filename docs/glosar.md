# Glosář — pojmy a zkratky

- **CMMS** — Computerized Maintenance Management System (systém řízení údržby).
- **WO / Work Order** — pracovní příkaz / úkol, kód `WO-2026-001`. Viz [[domena]].
- **Asset** — sledované zařízení/stroj, vázané na budovu a místnost. Viz [[tovarna]].
- **Kiosk** — režim pro tablet na zdi výrobní haly, role OPERATOR. Viz [[role]].
- **Revize** — zákonné/pravidelné kontroly (elektro, tlak, hasicí…), vázané na budovu.
- **ND** — náhradní díly (sklad ND = budova E).
- **Zóna** — `GLUTEN` / `GLUTEN_FREE` / `NEUTRAL`, kvůli křížové kontaminaci. Viz [[tovarna]].
- **P1–P4** — priority úkolu (P1 = havárie). Viz [[domena]].
- **PIN login** — 4místný kód mapovaný na `pin_XXXX@nominal.local`. Viz [[role]].
- **onSnapshot** — Firestore real-time listener; základ datového toku. Viz [[domena]].
